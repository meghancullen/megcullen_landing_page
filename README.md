# Meghan Cullen — Personal Landing Page

*This is a personal professional landing page. Replace the title with your own name and adjust the description to match your project.*

A static personal landing page built for **Meghan Cullen**, a Business Analytics and Marketing senior at the University of Iowa. The page exists to provide recruiters and hiring managers a dynamic, branded alternative to a traditional resume — showcasing Meghan's analytical projects, technical skills, and professional background in one polished, accessible location. The target audience is entry-level hiring teams at large global firms and mid-sized marketing agencies looking for data-driven candidates with a proactive mindset.

## Features

*List the key features of your project from a user's perspective. Update this list to reflect what your site actually does.*

- **Hero section** with name, professional tagline, and a circular headshot
- **About Me** section with a first-person bio drawn directly from the resume — no boilerplate
- **Skills** displayed as color-coded pill badges, separated into Technical Tools and Analytic Methods
- **Projects** section with three detailed cards describing real academic and professional work
- **Contact** section with labeled icon links to LinkedIn, GitHub, and UIowa email
- **Sticky navigation** bar with smooth-scroll anchor links to each section
- **Fully responsive** layout from 320 px mobile to wide desktop, with a two-column project grid on desktop
- **WCAG 2.2 Level AA** accessibility: descriptive alt text, logical heading hierarchy, keyboard navigation, and sufficient color contrast throughout

## Getting Started

### Prerequisites

*List everything someone needs installed before they can run this project locally.*

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- [Python 3](https://www.python.org/downloads/) — for running the local development server (comes pre-installed on macOS and most Linux systems)
- [Git](https://git-scm.com/downloads) — to clone the repository

No Node.js, npm, or build tools are required. This is a fully static site.

### Installation

*Update the repository URL in step 2 to match your own GitHub repo.*

1. Open a terminal.

2. Clone the repository:

```bash
git clone https://github.com/meghancullen/megcullen_landing_page.git
```

3. Navigate into the project folder:

```bash
cd megcullen_landing_page
```

4. Start the local development server:

```bash
python3 -m http.server 5000
```

5. Open your browser and go to:

```
http://localhost:5000
```

The landing page will load immediately — no build step required.

## Usage

*Describe how to use the application once it is running. Update this section if you add configuration options or data inputs.*

Once the server is running, open `http://localhost:5000` in your browser. Use the sticky navigation bar at the top to jump between sections: About, Skills, Projects, and Contact. The page is fully responsive — resize your browser window or view it on a mobile device to see the layout adapt. No login, configuration, or data entry is required.

To customize the page for your own use, edit `index.html` to replace the content (name, bio, skills, projects, contact links) and update `css/stylesheet.css` to adjust colors, fonts, or spacing. The headshot image is stored at `images/headshot_2025.jpeg` — replace that file with your own photo (keep the same filename, or update the `src` attribute in `index.html`).
